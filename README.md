# xib_Tool
### xib的工具类目。

* 1.可以在xib可视化的设置视图圆角，边框。<br> 
* 2.可以在xib设置单个约束等比自适应开关。（以6为基础，原型图通常为这个尺寸）<br> 
* 3.可以在xib设置按钮响应时间间隔等操作。<br> 
* 4.可以在xib设置文字下划线，删除线及其颜色。<br> 
* 5.可以在xib设置文字自适应缩放。（以6为基础，View+FontZoom扩展，控件+FontZoom扩展2选一）<br>
（后续还在更新ing。。。。）<br>
